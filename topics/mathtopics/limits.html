Function

Functions are one of the most fundamental ideas in mathematics. They describe how one quantity depends on another.

Why This Concept Exists

Mathematics often studies relationships between quantities. For example, distance may depend on time, or cost may depend on the number of items purchased.

Functions provide a formal way to describe such relationships.

Core Idea

A function is a rule that assigns each input exactly one output.

If we represent the function as f(x), the value of the function depends on the value of x.

History Corner

The concept of functions evolved gradually. Mathematicians such as Leonhard Euler played a major role in formalizing the notation and idea of functions.
